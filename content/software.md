### Videos(and related):
- [Kdenlive](https://kdenlive.org/) - main video editor
- [Smoothie-RS](https://github.com/Hzqkii/smoothie-rs) - motionblur
- [mpv](https://mpv.io/) - the best video player to date

### Photos:
- [GNU Image Manipulation Program(GIMP)](https://gimp.org/) - amazing photo editor
- [Gwenview](https://apps.kde.org/gwenview/) - photo viewer
- [DigiKam](https://digikam.org/) - album management
- [ImageMagick](https://imagemagick.org/) - cli tool to convert and edit images

### Audio:
- [SoX](https://sourceforge.net/projects/sox/files/) - terminal audio editor, useful for automation
- [Audacity](https://audacityteam.org/) - audio editor
- [Audacious](https://audacious-media-player.org/) - great music player

### Operating systems:
- [Artix](https://artixlinux.org/) - wonderful 1 week journey
- [Arch](https://archlinux.org) - bleeding edge, lots of packages
- [MX Linux](https://mxlinux.org/) - useful operating system with tools for a secondary computer
